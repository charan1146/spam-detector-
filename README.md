# AI Spam Email Detector

This project is a beginner-friendly spam email detection system developed using Python, Streamlit, and Scikit-learn.

The application predicts whether a message is spam or not spam using machine learning and natural language processing techniques.

## Features

- Detects spam and non-spam messages
- Simple web interface using Streamlit
- Uses machine learning for prediction
- Beginner-friendly implementation
- Uses a CSV dataset for training

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn

## Machine Learning Concept

This project uses:

- Natural Language Processing (NLP)
- Text Classification
- Naive Bayes Algorithm

The text data is converted into numerical format using CountVectorizer before training the model.
